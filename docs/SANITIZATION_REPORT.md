# Sanitization Report

This GitHub-ready version was prepared for public portfolio use.

## Removed

- Local `.venv` virtual environment
- `.idea` IDE configuration files
- `__pycache__` compiled Python files
- Original university-specific logo/image files
- RAR/ZIP source archive files
- Any public display of university identity or branding

## Kept

- Core Python source code
- Project logic and BFS implementation
- Hassan Abdulsalam Mohammed's name as project portfolio credit
- MIT license and attribution notice
- Clean README and documentation
- Neutral portfolio screenshots

## Reason

The goal is to keep the technical value of the project visible while avoiding exposure of private or institution-specific identity in a public GitHub repository.
